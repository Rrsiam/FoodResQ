# FoodResQ - Food Waste Management System

A comprehensive web-based platform to reduce food waste by connecting donors with NGOs and converting expired food into organic pesticides.

#Live Demo
https://rrsiam.github.io/FoodResQ/

## ✨ Features

### Core Features
- **Multi-User System**: 5 user types (Admin, Donor, NGO, Collector, Plant Operator)
- **Donation Management**: Track food donations from source to destination
- **Smart Routing**: Edible food → NGOs | Expired food → Processing Plants
- **Collection Scheduling**: Schedule and track pickups
- **Location Services**: Interactive maps with geocoding
- **Alert System**: Real-time notifications for users
- **Feedback System**: Rate and review system performance
- **Resource Tracking**: Monitor water, electricity, and gas usage
- **Waste Assessment**: Quality grading for waste materials

## 👥 User Roles

### 1. Admin
- Full system control
- Manage all users
- View all data
- Configure system settings
- Send alerts to anyone

### 2. Donor
- Register food donations
- Schedule pickups
- View donation history
- Receive alerts
- Submit feedback

### 3. NGO
- Receive food donations
- Manage food inventory
- Track collection schedules
- View donation reports
- Provide feedback

### 4. Collector
- View assigned pickups
- Update collection status
- Navigate to locations
- Receive alerts
- Submit reports

### 5. Plant Operator
- Process expired food
- Track resource usage
- Manage waste quality
- Monitor plant operations
- Record production data
